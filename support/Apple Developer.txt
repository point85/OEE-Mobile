How To Install Flutter:
Step-by-Step Installation
1. Download the Flutter SDK
Go to the Flutter macOS install page: https://docs.flutter.dev/get-started/install/macos

Download the appropriate version for your chip (Apple Silicon or Intel)

Extract it to a directory like ~/Development/flutter

2. Add Flutter to Your PATH
Open Terminal and edit your shell config file:

For Zsh: nano ~/.zshrc or nano ~/.zshenv

For Bash: nano ~/.bash_profile

Add this line (adjust the path if needed): 

bash
export PATH="$PATH:$HOME/Development/flutter/bin"
Save and restart Terminal

3. Run Flutter Doctor
bash
flutter doctor
This checks your environment and shows what’s missing (like Xcode or Android Studio).

4. Install Xcode (for iOS)
Download from the App Store

Open Xcode → Preferences → Locations → Set Command Line Tools

5. Install Android Studio (for Android)
Download from developer.android.com

During setup, install the Flutter and Dart plugins

✅ Optional: Use Rosetta 2 (for Apple Silicon)
If you're on an M1/M2/M3 Mac, install Rosetta:

bash
softwareupdate --install-rosetta --agree-to-license
